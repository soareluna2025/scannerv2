2.1 GET /fixtures — Câmpuri IGNORATE
────────────────────────────────────────────────────────────────────────────────

  Din obiectul fixture{}:
    ✗ fixture.referee           — Numele arbitrului
    ✗ fixture.venue.id          — ID stadion (util pentru meteo via coordonate!)
    ✗ fixture.venue.name        — Nume stadion
    ✗ fixture.venue.city        — Oraș (util pentru meteo.js)
    ✗ fixture.timezone          — Fus orar
    ✗ fixture.timestamp         — Unix timestamp
    ✗ fixture.periods.first     — Timestamp start prima repriză
    ✗ fixture.periods.second    — Timestamp start a doua repriză
    ✗ fixture.status.extra      — Minute de prelungiri acordate

  Din obiectul score{}:
    ✗ score.halftime.home/away  — Scor la pauză ⚠️ IGNORAT — util pentru predicții!
    ✗ score.extratime.home/away — Scor prelungiri
    ✗ score.penalty.home/away  — Scor penaltiuri

  Din obiectul league{}:
    ✗ league.round              — Etapa curentă (ex: "Regular Season - 28")
    ✗ league.season             — Sezon (ex: 2025)
    ✗ league.logo               — Logo ligă

  Din obiectul teams{}:
    ✗ teams.home.logo / away.logo
    ✗ teams.home.winner / away.winner — true/false/null (câștigător)
