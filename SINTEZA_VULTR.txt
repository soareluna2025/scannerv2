3. PROBLEME INTAMPINATE SI REZOLVARILE
----------------------------------------------------------------

PROBLEMA: Port 22 blocat din container Claude
SOLUTIE:  GitHub Actions SSH via appleboy/ssh-action

PROBLEMA: SSH key corupta la copy-paste pe mobil
SOLUTIE:  Autentificare cu parola in loc de cheie SSH

PROBLEMA: Node.js v12 in loc de v20
SOLUTIE:  snap install node --classic --channel=20

PROBLEMA: PM2 trimitea SIGTERM si oprea sesiunea SSH
SOLUTIE:  Inlocuit PM2 cu systemd service

PROBLEMA: Heredoc in YAML rupea parsing-ul workflow-ului
SOLUTIE:  Toate fisierele scrise cu python3 string literals

PROBLEMA: Port 80 ocupat de Coolify (pre-instalat de Vultr)
SOLUTIE:  docker stop + fuser -k 80/tcp inainte de nginx

PROBLEMA: Cheile API lipseau din .env
SOLUTIE:  GitHub Secrets - scrise automat la fiecare deploy
