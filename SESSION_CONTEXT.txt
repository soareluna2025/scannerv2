A. POISSON 6Г—6 (calc-utils.js вҶ’ calcPoisson6x6)
   Matrix 6Г—6 scoruri (0-0 pГўnДғ la 5-5):
     P(i,j) = Poisson(О»H, i) Г— Poisson(О»A, j)
   Derivate:
     homeWin = ОЈP(i>j), draw = ОЈP(i=j), awayWin = ОЈP(i<j) вҶ’ normalizate la 100%
     over15 = ОЈP(i+jвүҘ2), over25 = ОЈP(i+jвүҘ3), ggProb = ОЈP(i>0 AND j>0)

   Lambda calcul (enrich.js):
     О»H = (homeAvgScored + awayAvgConceded) / 2   вҶҗ cross-product model corect
     О»A = (awayAvgScored + homeAvgConceded) / 2
   DacДғ date insuficiente: lgHome (avg_home_goals din league_stats) sau 1.2

   Dynamic lambda (mecuri live):
     minutesLeft = 90 - elapsed
     intensityFactor = 1 + min(SOT/25*elapsed*90вҒ»В№, 0.4)
     О»_remaining = О»Base Г— (minutesLeft/90) Г— intensityFactor
     О»_dynamic = currentGoals + О»_remaining

B. EXPECTED VALUE (calc-utils.js вҶ’ calcEV)
   impliedH = 1/cotaH, impliedD = 1/cotaD, impliedA = 1/cotaA
   totalImplied = impliedH + impliedD + impliedA
   evHome = (homeWin/100) - (impliedH / totalImplied)
   evOver15 = (over15Prob/100) - (1/cotaOver15)
   evGG     = (ggProb/100) - (1/cotaGG)

   Kelly:
     kelly = min(bankroll Г— edge Г— 0.5, bankroll Г— 0.04)   dacДғ edge > 4%
     altfel: kelly = 0

C. CONFIDENCE SCORE 7 STRATURI (enrich.js вҶ’ calcConfidence)

   Layer 1 вҖ” Poisson over15Prob (22%)
   Layer 2 вҖ” FormДғ recentДғ: min(100, (homeAvg+awayAvg)/3.5*100) (20%)
   Layer 3 вҖ” H2H over15% (10%)
   Layer 4 вҖ” Live xG/SOT/DA: xgГ—25 + sotГ—3 + daГ—0.5 (15%)
              вҶ’ dacДғ SOT=0 dupДғ min10: score4 = 20 (penalizare)
   Layer 5 вҖ” EV: best market EVГ—300 (8%)
   Layer 6 вҖ” ConsistenИӣДғ: (layere>60 / 5) Г— 100 (5%)
   Layer 7 вҖ” Puterea echipei din player_stats (20%) вҖ” activ dacДғ вүҘ10 jucДғtori/echipДғ

   PenalizДғri runtime:
     elapsed < 15min: score Г— 0.85
     elapsed 15-30min + SOT=0: score Г— 0.80
     elapsed вүҘ 45min + SOT=0: score - 20
     yellow_cards вүҘ 2: score - 10
     injuries вүҘ 3/5/8: score - 5/10/15

F. NGP вҖ” Next Goal Probability (scan.js / scanner.js)

G. MONTE CARLO (monte-carlo.js вҶ’ runSimulation)
